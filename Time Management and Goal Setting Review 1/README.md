# TimeTrack Application

## Backend Setup

1.  **Database**: Make sure you have MySQL running. Create a database named `timetrack`.
2.  **Schema**: Run the `backend/src/main/resources/schema.sql` script to create the tables.
3.  **Data**: Run the `backend/src/main/resources/sample-data.sql` script to populate the database.
4.  **Configuration**: Update the `backend/src/main/java/com/timetrack/util/DBConnectionManager.java` with your MySQL username and password.
5.  **Run**: Run the `Main.java` file in the backend to start the server.

## Frontend Setup

1.  **Serve**: Open the `frontend/index.html` file in your browser.
